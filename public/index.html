This index.html is for testing purposes, served at localhost:5000. This will not show up in production, only the components showcased below.

"Back end" app

This is the component that ends up on the individual "counter" pages. In this example, I've added the location prop to the html element and passed in "gorgas". The backend prop is set to true, and the capacity prop is where you set the total allowable occupancy for the location.

<covid-counter-app backend="true" location="gorgas" capacity="200"></covid-counter-app>


"Front-end" app

This component is for public comsumption, so it's much simpler, no props to set up.

<covid-counter-app></covid-counter-app>
